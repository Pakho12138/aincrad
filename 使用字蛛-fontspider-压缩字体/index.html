# 使用字蛛（FontSpider）压缩字体

# 一、font-spider是什么？

通过分析本地 CSS 与 HTML 文件获取 WebFont 中没有使用的字符，并将这些字符数据从字体中删除以实现压缩，同时生成跨浏览器使用的格式。中文 WebFont 自动化压缩工具。官方网站 font-spider (opens new window)

# 字体下载推荐

字客网 (opens new window)

# 二、使用步骤

# 1.安装库

代码如下（示例）：

npm install font-spider -g
1

或者

yarn add font-spider -g
1

# 目录结构

# 编写css和html

注意：body中的内容就是我们要抽出为字体！！！

<!DOCTYPE html>
<html lang="en">
  <head>
    <meta charset="UTF-8" />
    <meta name="viewport" content="width=device-width, initial-scale=1.0" />
    <title>Document</title>

    <style>
      /*声明 WebFont*/
      @font-face {
        font-family: 'PingFang SC';
        src: url('./PingFang SC.eot');
        src: url('./PingFang SC.eot?#font-spider') format('embedded-opentype'), url('./PingFang SC.woff') format('woff'), url('./PingFang SC.ttf') format('truetype'),
          url('./PingFang SC.svg') format('svg');
        font-weight: normal;
        font-style: normal;
      }

      /*使用选择器指定字体*/
      body {
        font-family: 'PingFang SC';
      }
    </style>
  </head>
